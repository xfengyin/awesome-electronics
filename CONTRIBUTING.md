# Contributing to Awesome Electronics

Thank you for your interest in contributing to Awesome Electronics!

## Contribution Standards

### Criteria for Adding New Resources

1. **Relevance** - Resource must be directly related to electronics engineering
2. **Open Source Preferred** - Prioritize open source or free resources
3. **Active Maintenance** - Project must be actively maintained
4. **Valid Links** - All links must be accessible (please verify before submitting)
5. **Concise Description** - Description should not exceed 20 words

## Pull Request Process

1. Fork the repository
2. Create a new branch for your changes
3. Add your resource in the appropriate category
4. Submit a Pull Request

## Categories

Please add resources to the appropriate category:
- `Learning Resources`
- `Circuit Design (Schematic & PCB)`
- `Simulation & Modeling`
- `Component Libraries`
- `PCB Manufacturing`
- `Embedded Development`
- `Testing & Measurement`
- `Open Source Hardware`
- `Chinese Ecosystem`

## Chinese Resources

For Chinese resources, please add them to the `Chinese Ecosystem` section with Chinese descriptions.
