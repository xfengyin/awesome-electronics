# CLAUDE.md - AI Assistant Guidelines

This file provides guidance to AI assistants on how to work with this repository.

## Project Overview

**awesome-electronics** is a curated list of awesome resources for Electronic Engineers and hobbyists. This fork maintains both English and Chinese versions.

## Repository Structure

```
awesome-electronics/
├── README.md              # English version
├── README_zh.md           # Chinese version (中文版)
├── CONTRIBUTING.md        # Contribution guidelines
└── .github/
    └── ISSUE_TEMPLATE/
        └── resource-suggestion.md
```

## Key Categories

1. **Learning Resources** - Tutorials, courses, theory
2. **Circuit Design** - Schematic & PCB tools (EDA)
3. **Simulation & Modeling** - Circuit simulators
4. **Component Libraries** - Parts search & libraries
5. **PCB Manufacturing** - Fab houses & assembly
6. **Embedded Development** - Dev boards & kits
7. **Testing & Measurement** - Test equipment
8. **Open Source Hardware** - Project sharing platforms
9. **Chinese Ecosystem** - Chinese tools & resources

## AI Collaboration Guidelines

### When Adding Resources

1. Always verify links are working
2. Keep descriptions concise (under 20 words)
3. Place in appropriate category
4. Add Chinese resources to `Chinese Ecosystem` section
